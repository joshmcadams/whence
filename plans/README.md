## Findings considered and rejected (do not re-audit)

- `doctor` runs docker and socket queries sequentially: rarely-run diagnostic; not worth doing.
- `listOnce` runs `inventory.View` twice to count hidden rows: O(n) over tens of rows; not worth doing.
- CI cross-build matrix caching: builds take seconds for this codebase; not worth doing.
- gopsutil v4.26.5 → v4.26.6 patch bump: no known cost; fold into any future dep refresh.
- Old indirect `golang.org/x/text v0.3.8` and the three text-width indirects: forced by the charm v1 tree; resolved by plan 022.
- Deleting `internal/cli/collect.go` (audit DEBT-06, first half): superseded — plan 013 repurposes it as the CLI test seam.
- Kill-climb behavior change to hop through npm-interposed shells (audit CORRECTNESS-09): deferred; plan 008 fixes the docs to describe actual behavior instead. Revisit only as a deliberate invariant change with its own review.
- macOS notarization/signing: deferred (needs an Apple Developer ID); plan 011 removes the quarantine-strip hook and documents the Gatekeeper prompt instead.
- UDP scanning, daemon/persistence, launcher list changes, docker error strictness: decided tradeoffs recorded in AGENTS.md/DESIGN.md; out of scope.
- `execx` deadline-vs-exit boundary race: theoretical; no observable failure mode found.
- `classify` substring hints over-matching (e.g. `"air"`): nudges a confidence score only; not a correctness bug.
- bubbles `SetRows` stale-cursor panic and bubbletea confirm-prompt cropping: investigated and disproven during the audit.
